# 搜狗微信校招监控实测报告 (v28)

> 2026-07-09 实测：搜狗微信搜索能稳定拿到**搜索结果列表**（标题+公众号+摘要+时间），但**真实文章 URL/全文**会被 IP 级反爬挡住。

---

## 核心结论

| 数据 | 是否能拿到 | 备注 |
|---|---|---|
| **搜索结果列表** | ✅ 完全能 | 10 条/页，含标题+公众号+摘要+时间 |
| **真实文章 URL** | ❌ 触发反爬 | 需要点字验证码（"请依次点击【憾,久,惯,崎】"） |
| **文章全文** | ❌ 拿不到 | 同上 |
| **公众号主页** | ⚠️ 部分能 | 同样反爬 |
| **多页抓取** | ✅ 能 | 加 delay 2.5s, 抓 5 页 = 50 条数据 |

**实用结论**：搜狗微信**摘要+标题+公众号名**已经覆盖 80% 校招情报需求 — 看摘要就能判断是不是你要的内推/岗位信息。

## 反爬实测

**触发条件**：连续抓 10-20 个文章 URL 链接后，搜狗会返回：

```
IP: 119.123.207.7
访问时间: 2026.07.09 17:16:47
VerifyCode: f3f8c9f73a2c
请依次点击【憾,久,惯,崎】
```

**反爬级别**：IP 级（同一 IP 多次访问锁死）

**绕过方法**：
- 切 VPN/代理 IP 池
- 降低抓取频率（每次 delay 30s+）
- 用浏览器手动过验证码
- 商业付费：新榜/西瓜数据 API

## 总结

**搜狗微信免费 + 实时 + 数据全**，但**有 IP 级反爬**。
**实际价值 = 摘要 + 标题 + 公众号名（足够决策）+ URL 列表（人工去看）**。

**推荐使用策略**：
- 一周跑 1-2 次（避开反爬）
- 抓列表摘要做关键词过滤
- 命中关键词的文章 → 浏览器人工点击看全文
- 配合 [新榜/西瓜] 做精读（如果要全文 + 历史数据）
