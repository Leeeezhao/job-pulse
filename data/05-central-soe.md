## 核心特征（与互联网公司不同）

| 维度 | 央国企 | 互联网/AI企业 |
|---|---|---|
| **招聘时间** | 8-9 月集中开启, 10 月笔试, 11-12 月面试 | 滚动招聘, 全年有 |
| **招聘数量** | 几百到 1-2 万 (单家) | 几十到几百 |
| **岗位类型** | 偏 IT 系统/数据/AI 平台建设 | 偏业务算法/前沿研究 |
| **薪资** | 总包 15-30w, 顶配 50w | 30-100w+ |
| **户口** | 几乎都给（北京/上海必给） | 不一定 |
| **稳定性** | 编内终身, 编外合同制 | 35 岁危机 |
| **技术栈** | Java/数据库/数据中台为主 | CV/NLP/RL/大模型前沿 |
| **学历要求** | 硕博优先, 部分卡 985/211 | 985 起步, 部分看产出 |
| **算法岗数量** | 每家 10-50 个 (相对少) | 每家 100-500 个 |
| **投递方式** | 自建网站 + 51job 转发 | 自建 + 主流招聘平台 |

## 实战策略

### 1. 投递顺序（按岗位质量）

```
第一梯队（最高性价比）：
  ├─ 头部券商 quant (中信/中金/华泰) → 50-100w
  ├─ 银行科技子公司 (建信/工银/招银网络) → 30-50w + 户口
  └─ 中科院系 (上海AI实验室/深圳先进院) → 30-50w + 编内

第二梯队（求稳）：
  ├─ 国网信通/中国电科院 → 25-40w + 户口
  ├─ 央移动/电信 AI 研究院 → 25-40w + 户口
  ├─ 航天/航空/船舶 研究院 → 25-40w + 户口 + 编内
  └─ 交易所/中证登/中债登 → 20-30w + 户口

第三梯队（求上岸）：
  ├─ 国有银行分行 → 15-25w + 户口
  ├─ 政策性银行 → 20-30w + 户口
  ├─ 能源行业 → 15-25w + 户口
  └─ 烟草 → 20-30w + 福利
```

### 3. 备考内容

**行测**（所有央国企通用）：
- 言语理解 (40 题)
- 数量关系 (15 题)
- 判断推理 (40 题)
- 资料分析 (20 题)
- 常识判断 (20 题)
- 申论 (国考/部分央企)

**专业知识**（按行业）：
- 银行：金融基础 + 经济 + 财会

## 总结

| 指标 | 数据 |
|---|---|
| 央国企总数 | 100+ 家有招聘 |
| 算法岗密集度 | 银行科技子公司 > 券商 quant > 中科院 |
| 顶级薪资 | 头部券商 quant 100w+ |
| 平均薪资 | 15-30w (vs 互联网 30-60w) |
| 户口概率 | 90%+ (vs 互联网 50%) |
| 稳定性 | 编内终身 (vs 35 岁危机) |
| 推荐顺序 | 银行科技子公司 ≥ 券商 quant > 中科院 > 央企研究院 > 国企分行 |

**实用结论**：**央国企是"求稳 + 户口 + 终身制"的最佳选择，但薪资天花板低于互联网**。最优策略：**秋招 8-10 月同时投央国企 + 互联网，根据 offer 情况决定**。
