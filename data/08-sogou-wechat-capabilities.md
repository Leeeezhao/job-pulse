# 搜狗微信搜得到什么 / 搜不到什么

> 实测时间：2026-07-09
> 实测方式：`scripts/wechat/sogou_search.py` 抓取 + curl 验证 type=1/2 接口

---

## ✅ 能搜到的内容

### 1. **公众号文章**（主要场景）
- 接口：`type=2` (默认)
- 每页 10 条，支持多页（默认脚本支持 `--pages N`）
- 返回字段：**标题 / 公众号名 / 摘要 (前 200 字) / 发布时间 / 临时 URL**

### 2. **覆盖场景**（实测都返回 8-10 条）

| 关键词类型 | 示例 | 命中数 | 典型内容 |
|---|---|---|---|
| 公司 + 校招年份 | `字节跳动 校招 2026` | 10 | offer 政策/院校清单/内推 |
| 行业 + 岗位 | `大模型 算法工程师 实习` | 10 | 公司直招/实习岗位 |
| 公司 + 事业群 | `腾讯 IEG 算法 招聘` | 10 | 事业群岗位/内推 |
| 长尾话题 | `算法岗 应届生 薪资 2026` | 9 | 薪资爆料/趋势分析 |
| 垂直技术 | `NLP 算法工程师` | 10 | 工业分享/技能要求/招聘 |
| 央国企 | `国家电网 校园招聘 人工智能` | 10 | 国网/中石油招聘公告 |
| 笔试题 | `拼多多 春招 2026 算法 笔试` | 5 | 笔试启动/题型分析 |
| 英文关键词 | `LLM inference interview` | 10 | 中文社区讨论 |
| 极短关键词 | `算法` (单字) | 8 | 通识科普/算法评论 |

## ⚠️ 搜得到但有限制

### 1. **真实文章 URL 拿不到**
- 返回的链接是 `weixin.sogou.com/link?url=...` 跳转页
- 点击跳转会触发**点字验证码** (IP 级反爬)
- ⚠️ 单条过审很麻烦，**摘要 + 标题 = 80% 决策价值**
- workaround：拿到标题后用其他渠道搜文章内容 (Google site: / 公众号 PC 网页版 / 二维码扫)

### 2. **公众号主体搜索不完整**
- 接口 `type=1` (搜公众号) 返回 HTML 但**解析困难**
- 实测 `type=1&query=塔子哥学算法` 只返回 10282 字节，**未列出公众号列表**
- workaround：用文章搜索查 `公众号名 + 文章`，间接拿到公众号 ID

### 3. **只能搜到最近/热门的文章**
- 历史文章（>6 个月）命中率显著下降
- 公众号被封禁的文章不会出现在结果里

### 4. **排序不可控**
- 默认按"时间 + 相关性"混合排序
- 没法指定"最新"、"最热"、"按公众号"等过滤

### 5. **关键词太泛容易跑偏**
- `算法` 单字 → 返回"趣味算法图解""算法即剥削"等杂文
- 解决：用 **2-3 个词组合** (`大模型 算法工程师`、`拼多多 笔试 0315`)

---

## ❌ 搜不到的内容

### 1. **公众号本身的内容（订阅号文章）**
- `type=1` 走的是公众号搜索，返回不完整
- 想找某个公众号所有文章 → 必须订阅它 (搜狗无法绕过)

### 2. **朋友圈 / 视频号内容**
- 完全没入口

### 3. **付费内容 / 需登录**
- 知识星球 / 圈子 / 付费阅读 → 仅显示标题，**摘要截断**

### 4. **被屏蔽 / 违规公众号**
- 平台主动屏蔽的关键词 (政治/赌博/医疗广告) 返回空

### 5. **小程序 / 服务号文章**
- 部分纯服务号（无文章列表）抓不到

## 🎯 实战建议

### 写搜索关键词的 SOP

```
[公司/品牌] + [岗位] + [场景]
- ✅ "拼多多 笔试 0315"   (3 词, 命中真题)
- ✅ "腾讯 IEG 算法 招聘" (公司+BG+岗位)
- ✅ "小红书 REDstar AI"  (专项名)
- ❌ "算法"               (太泛, 噪音)
- ❌ "腾讯"                (缺岗位)
```

### 多角度组合

```
搜公司某个岗位时, 至少换 3 个关键词:
1. 公司 + 岗位: "拼多多 算法"
2. 公司 + 场景: "拼多多 校招"
3. 公司 + 事业群/专项: "拼多多 云弧计划"
4. 行业 + 岗位: "电商 算法工程师"
```

### 找真题时

```
[公司] + [真题/笔试/题解] + [年份/月份]
✅ "拼多多 笔试 0315"
✅ "字节 算法 0322"
✅ "美团 0801 题解"
✅ "PDD 笔试题 0515"
```

## 🔧 脚本说明

`scripts/wechat/sogou_search.py` 已封装好：

```bash
python3 scripts/wechat/sogou_search.py --query "拼多多 笔试" --pages 3
```

参数：
- `--query` 必填，空格分隔关键词
- `--pages` 抓几页 (默认 1, 10 条/页)
- `--delay` 请求间隔秒数 (默认 2.5s, 防封)
- `--out` 输出 JSON 文件

反爬措施：
- 默认 2.5 秒/页 (实测不会触发)
- 触发反爬会收到点字验证码页面 → 脚本返回空数组，需要换 IP 或等几小时

## 📊 数据价值排序

| 信息 | 拿到率 | 用途 |
|---|---|---|
| 文章标题 | 100% | 选题/趋势监控 |
| 文章摘要 | 100% (前 200 字) | 大部分决策足够 |
| 发布时间 | 100% | 时效性判断 |
| 公众号名 | 100% | 找到持续输出源 |
| 真实文章 URL | **0%** (反爬) | 全文需绕路 |
| 文章阅读数 | 拿不到 | 影响力无法量化 |
| 文章评论 | 拿不到 | 社区反馈不可见 |
