### 按学历投递策略

#### 博士
1. 顶会论文是硬通货（NeurIPS / ICML / ICLR / CVPR / ACL）
2. 优先投递 **Research Scientist** 类岗位
3. 海外 PhD 推荐 Meta FAIR / Google Research / OpenAI / Anthropic
4. 国内 Top PhD 推荐 达摩院 / 字节 AI Lab / MSRA / DeepSeek

#### 硕士
1. 算法岗简历看重 **顶会二作 + 大厂实习 + 强项目**
2. 优先投递 **Applied Scientist / 算法工程师** 类岗位
3. 实习转正是主路径（>50% 转正率）
4. 海投 + 强内推是核心策略

#### 本科
1. ACM 竞赛 / Kaggle / 开源贡献是加分项
2. 主要走 **实习转正** 路径
3. 秋招投递大厂的机会有限，建议提前实习

### 实习 vs 校招 时间规划

```
大三上学期 (9-12 月)
├─ 准备：刷题（LeetCode 200 题）+ ML/DL 基础 + 项目整理
├─ 投递：日常实习
└─ 目标：积累 1 段实习

大三下学期 (3-6 月)
├─ 投递：暑期实习（关键！）
├─ 准备：算法岗面试 + 系统设计
└─ 目标：拿到大厂暑期实习 Offer

大三暑假 (7-9 月)
├─ 实习：核心业务团队
├─ 准备：秋招提前批
└─ 目标：转正答辩 + 秋招提前批 Offer

大四上学期 (9-11 月)
├─ 秋招：正式批 + 补录
├─ 准备：面试复盘
└─ 目标：拿到心仪 Offer

研究生阶段同理，往后推迟 1-2 年
```

## ⚠️ 重要提示

### 行业趋势（2026）

1. **LLM 已成红海**：所有大厂都在招 LLM 算法，竞争激烈
2. **多模态 + Agent 是新方向**：未来 2-3 年的增长点
3. **AI Infra 紧缺**：训练/推理优化工程师需求大
4. **端侧 AI 兴起**：面壁智能 / 高通 / 苹果都在布局
5. **CV 行业收缩**：传统 CV 岗位减少，建议结合多模态背景
6. **自动驾驶进入 L3+**：感知/规划岗位需求增加
7. **机器人元年**：人形机器人方向值得关注

### 投递避坑

1. **不要海投无脑**：每个岗位都要针对性修改简历
2. **不要只投大厂**：保底 offer 很重要
3. **不要错过提前批**：提前批竞争小、机会多
4. **不要忽略内推**：内推简历优先筛选
5. **不要只做执行**：实习要主动思考业务逻辑
6. **不要隐瞒经历**：诚信是底线

### 技能补充建议

1. **刷题**：LeetCode 300 题 + 剑指 Offer
2. **ML 基础**：西瓜书 / 李航统计学习方法 / 花书
3. **DL 实战**：李沐《动手学深度学习》/ PyTorch 官方教程
4. **论文阅读**：顶会论文精读 50 篇 + 泛读 200 篇
5. **开源贡献**：参与知名开源项目（HuggingFace / MegEngine / MMlab）
6. **竞赛**：Kaggle / 天池 / 阿里云 / KDD Cup
7. **英语**：口语流利 + 听力过关
