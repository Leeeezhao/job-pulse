# job-pulse — 2026 校招算法岗情报站

> **v3：基于真实浏览器实测的校招投递链接**
> **实测日期**：2026-07-07
> **方法**：用 Playwright 启动 Chromium，对每家公司执行"打开主页 + 搜索方向词 + 验证 URL 是否生效"

## ⚠️ 重要前提

**v1 / v2 的链接大量是瞎写的**——我假设所有公司都支持 `?keywords=方向` 这种 URL 搜索参数，但**实测证明大多数公司不支持**（URL 参数被服务器忽略，跳默认列表）。

**v3 重新整理**，明确区分：

| 类型 | 含义 | 公司 |
|---|---|---|
| 🔍 **URL 搜索有效** | URL 参数真的过滤了岗位 | 字节 / 美团 / 腾讯 / 知乎 / 智谱 / 百川 / 月之暗面 / MiniMax |
| 🏠 **主页 + 站内搜索** | 必须进首页用站内搜索框 | 阿里 / 百度 / 京东 / 华为 / 拼多多 / 滴滴 / 快手 / OPPO / vivo / 大疆 / 小米 / 蔚来 / 小鹏 / 理想 / 小红书 |

## 💡 给用户的使用建议

1. **优先点这 4 家公司的链接**（URL 搜索真实有效）：
   - 字节 / 美团 / 腾讯 / 知乎
2. **其余公司**：点主页 → 用站内搜索框搜方向词 → 选"职位类别"过滤
3. **不要相信一个看起来正常的 URL 真的过滤了关键词**——很多公司的 URL 参数被忽略

## 📊 v3 vs v2 改进

| 维度 | v2 | v3 |
|---|---|---|
| 链接来源 | 瞎猜 + curl HEAD 验 200 | Playwright 实测 + 验证内容 |
| 字节 / 美团 / 腾讯 | ✅ URL 搜索有效 | ✅ URL 搜索有效（实测） |

| 阿里 / 拼多多 | ⚠️ URL 瞎写 | ❌ 实测无效，标注为"主页+站内搜索" |
| OPPO / 大疆 / 小红书 | ⚠️ URL 瞎写 | ✅ 实测找到校招子站 URL |
