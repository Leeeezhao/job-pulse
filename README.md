| **招聘对象** | 2026 届（毕业时间） |

| **流程** | 网申→笔试→面试→Offer |
| **JD 关键要求** | 编程语言/算法方向/项目经验 |
| **投递链接** | 带方向关键词的搜索 URL（点进去直接看到该方向所有岗位） |
| **内推码** | （如有） |
| **公众号** | 招聘公众号 |
| **来源** | 数据来源 URL |

## ⚠️ 注意事项

1. **链接时效性**：官网 URL 可能因公司改版而失效，访问失败请尝试搜索引擎最新链接
2. **内推码**：仅作参考，实际投递时建议联系在职学长学姐获取最新内推
3. **截止日期**：大部分公司秋招 9-10 月截止，春招 4-6 月截止；提前批越早越好
4. **数据更新**：本目录为一次性快照，建议每季度更新一次
5. **招聘信息真假**：优先认准官方渠道（官网 + 招聘公众号），第三方信息需交叉验证
