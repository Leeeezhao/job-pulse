## 💡 给用户的使用建议

**优先级**：
1. **✅ 优先用这 5 家**：字节 / 美团 / 腾讯 / 知乎 / vivo 的链接（URL 搜索真实有效）
2. **🏠 这 9 家**：进首页 → 用站内搜索框搜方向词
3. **❌ 这 4 家**：放弃 URL 搜索，直接进首页

**关键经验**：
- 不要相信一个看起来正常的 URL 真的过滤了关键词
- 阿里 / 拼多多 / 蔚来 / 滴滴的 `?keyword=X` 是无效的
- vivo 的 URL 不变但 body 内容确实过滤了——这是前端 JS 搜索
