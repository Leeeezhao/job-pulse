## 🎯 算法岗高频考点

### 数据结构
- 数组 / 字符串（高频）
- 链表（中等）
- 树（二叉树 + 字典树，中等）
- 堆 / 优先队列（中频）
- 并查集（偶尔）
- 单调栈 / 单调队列（高频）

### 算法
- **二分答案** ⭐⭐⭐⭐⭐（拼多多最爱）
- **模拟** ⭐⭐⭐⭐⭐
- **动态规划** ⭐⭐⭐⭐
- **贪心** ⭐⭐⭐⭐
- **前缀和 / 差分** ⭐⭐⭐
- **图论（Dijkstra / BFS / DFS）** ⭐⭐⭐
- **数学 + 位运算** ⭐⭐

### SQL（数据岗重点）
- 多表 JOIN + 聚合
- 窗口函数 ROW_NUMBER / RANK
- 时间函数 + 业务分析
- 笛卡尔乘积去重

1. **拼多多题库部分来自清华/南大考研真题**，建议刷 LeetCode 时注意高频题
